## 🧠 Memory and Lore

This software uses a knowledge graph to track the history of your roleplay sessions. Unlike standard chatbots, this system remembers events from weeks ago. It adjusts the game world based on your actions. If you save a village in one session, the residents will treat you like a hero in the next session. You do not have to remind the AI about previous events. It retrieves this information automatically.

## 🛠 Features

*   **Persistent Memory:** The system stores every interaction in a database. Your story continues exactly where you left it.
*   **Evolving Lore:** The application tracks facts about your world. As you discover new places, the system updates its database to reflect these changes.
*   **Autonomous World Updates:** The game world reacts to your choices without you needing to prompt the AI. Non-player characters make decisions based on their personality and history.
*   **Context Retrieval:** The AI scans your past interactions to provide accurate and relevant responses. It understands complex plot threads and long-term narrative arcs.
*   **Privacy:** All processing happens on your local machine. Your logs and character files never leave your computer.

## 🛡 Frequently Asked Questions

**Does this software require an internet connection?**
No. Once you download the software, it functions entirely offline. You only need the internet once to check for future updates.

**Can I export my story logs?**
Yes. Every session creates a text file containing the full transcript of your roleplay. You find these in the Logs folder within the application directory.

**How do I clear the memory?**
If you want to start a fresh adventure, click the Refresh icon in the main menu. This clears the active knowledge graph and resets the persistent memory for the current profile.

**Can I use my own character sheets?**
Yes. You can import character text files into the Profiles section. The AI reads these files to understand your character traits, inventory, and background history.

**What happens if the application crashes?**
The system saves your progress after every turn. If the application closes, reopen the program and select Continue Session from the main landing page. It reloads your state from the last saved point.

## 📝 Troubleshooting

If you encounter issues during startup, check these common items:

*   **Antivirus Software:** Sometimes security software blocks local AI tools. If the application fails to launch, add the application folder to your antivirus exclusion list.
*   **Graphics Drivers:** Ensure your graphics drivers are up to date. Visit the website of your graphics card manufacturer to download the latest version.
*   **Disk Space:** Ensure at least 10GB of free space on your drive. The software needs this room to store the local database and model files.
*   **System Restart:** A simple reboot often fixes minor memory conflicts on Windows. 

If these steps do not fix your issue, check the issues tab on the repository to see if others encounter the same trouble. Provide your system details and a description of the error when posting a request for help.
